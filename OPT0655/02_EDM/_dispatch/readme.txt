How to edit and send an .eml file in Outlook

1. Open the .eml file
Double-click the file or right-click and choose Open with > Outlook. The draft message will appear.

-----

2. Click 'Forward'
In the toolbar, hit Forward. This gives you a clean version to send.

-----

3. Add send details

Enter the recipient’s email address in the To field.

-----

4. Update the Subject if required.

-----

5. Remove forwarding artefacts
Delete any extra text like:

“---------- Forwarded message ---------”

Previous sender info
Quoted content you don’t want to keep

-----

6. Edit the content
Make your changes directly in the email body. Fix typos, update links, swap out content—whatever is needed.

-----

7. Send
Review, proofread, then hit Send.
